# 📌 Introdução aos Sistemas Operacionais

Os sistemas operacionais são responsáveis por gerenciar os recursos do computador e permitir a comunicação entre hardware, software e usuário.

Eles controlam:
- Processador;
- Memória;
- Arquivos;
- Dispositivos;
- Aplicativos;
- Usuários e permissões.

Os sistemas operacionais evoluíram para atender diferentes necessidades computacionais, desde grandes centros de processamento até computadores pessoais e servidores modernos.

---

# 🖥️ Tipos de Sistemas Operacionais

## 🏢 Sistemas de Grande Porte (Mainframes)

Os sistemas mainframe são projetados para processamento massivo de dados e alta capacidade de Entrada/Saída (E/S).

São utilizados em:
- Bancos;
- Grandes varejistas;
- Companhias aéreas;
- Sistemas críticos governamentais.

## Características principais

- Alta disponibilidade;
- Segurança elevada;
- Processamento de milhares de transações;
- Suporte a múltiplos usuários simultaneamente;
- Grande confiabilidade operacional.

## Exemplos

- OS/360
- OS/390
- Linux para mainframes
- UNIX

---

# 🌐 Sistemas Operacionais de Servidor

Os sistemas operacionais de servidor são desenvolvidos para compartilhar recursos e serviços em rede.

Esses sistemas oferecem:
- Hospedagem de sites;
- Bancos de dados;
- Compartilhamento de arquivos;
- Controle de usuários;
- Serviços corporativos.

## Principais características

- Estabilidade;
- Escalabilidade;
- Segurança;
- Gerenciamento remoto;
- Alto desempenho.

## Exemplos

### Linux Server
Muito utilizado em:
- Cloud Computing;
- Data Centers;
- Hospedagem web;
- Infraestrutura corporativa.

### Windows Server
Muito utilizado em:
- Active Directory;
- Redes corporativas;
- Serviços Microsoft;
- Gerenciamento empresarial.

# 🔄 Controle de Versão com Git

O Git é um sistema de controle de versão utilizado para registrar alterações em projetos.

Ele permite:
- Salvar versões do projeto;
- Trabalhar em equipe;
- Recuperar alterações;
- Organizar o desenvolvimento.

---

# 🚀 GitHub

O GitHub é uma plataforma baseada em Git utilizada para armazenar repositórios online.

## Benefícios

- Compartilhamento de projetos;
- Versionamento;
- Portfólio profissional;
- Colaboração em equipe;
- Backup do código.

---

# ✅ Boas Práticas de Versionamento

## Commits organizados

Cada alteração deve possuir uma descrição clara.

### Exemplos

- Atualiza documentação
- Corrige erros
- Adiciona atividade
- Melhora estrutura do projeto

# 📌 Conclusão

Os sistemas operacionais possuem papel fundamental na computação moderna, sendo responsáveis pela administração dos recursos computacionais e pela execução eficiente de aplicações.

Além disso, ferramentas como Git e GitHub se tornaram essenciais para organização, desenvolvimento e versionamento de projetos tecnológicos.

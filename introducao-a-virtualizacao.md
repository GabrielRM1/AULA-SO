# 📌 Introdução

A virtualização é uma tecnologia utilizada para criar ambientes computacionais virtuais dentro de um único computador físico.

Ela permite executar múltiplos sistemas operacionais simultaneamente, compartilhando os recursos do hardware de forma eficiente e segura.

Atualmente, a virtualização é amplamente utilizada em:
- Data centers;
- Computação em nuvem;
- Desenvolvimento de software;
- Ambientes corporativos;
- Laboratórios de testes.

---

# 🖥️ Definição de Virtualização

A virtualização consiste na criação de máquinas virtuais capazes de simular um computador real.

Cada máquina virtual possui:
- Sistema operacional próprio;
- Memória virtual;
- Processador virtual;
- Disco virtual;
- Configurações independentes.

Isso permite que diferentes sistemas operacionais funcionem simultaneamente no mesmo hardware físico.

---

# ⚙️ Conceito de Máquina Virtual

Uma máquina virtual (VM) é um ambiente isolado criado por software que simula completamente um computador físico.

As máquinas virtuais podem executar:
- Windows;
- Linux;
- macOS;
- Sistemas servidores;
- Ambientes de testes.

Cada VM funciona independentemente do sistema operacional principal.

---

# 🔄 Funcionamento da Virtualização

A virtualização utiliza um software chamado hipervisor.

O hipervisor é responsável por:
- Distribuir recursos do hardware;
- Criar máquinas virtuais;
- Controlar memória;
- Gerenciar processamento;
- Isolar os ambientes virtuais.

---

# 🧠 Tipos de Virtualização

## Virtualização de Hardware

Simula computadores completos utilizando recursos físicos compartilhados.

## Virtualização de Servidores

Consolida múltiplos servidores em um único hardware físico.

## Virtualização de Desktop

Permite acessar sistemas operacionais remotamente.

## Virtualização de Rede

Cria redes virtuais independentes.

---

## 🌐 Execução de Múltiplos Sistemas

Permite utilizar diferentes sistemas operacionais simultaneamente.

---

## ☁️ Base da Computação em Nuvem

A virtualização é uma das tecnologias fundamentais do Cloud Computing.

# 🖥️ O que é Oracle VirtualBox?

O Oracle VirtualBox é um software gratuito de virtualização utilizado para criar máquinas virtuais.

Ele permite instalar diferentes sistemas operacionais dentro de um computador físico.

## Principais características

- Gratuito;
- Compatível com Windows, Linux e macOS;
- Fácil criação de VMs;
- Suporte a snapshots;
- Ambientes isolados para testes.

---

# 🛠️ Criando uma Máquina Virtual

## Etapas básicas

1. Instalar o VirtualBox;
2. Baixar a imagem ISO do sistema operacional;
3. Criar uma nova máquina virtual;
4. Definir memória RAM e armazenamento;
5. Iniciar a instalação do sistema operacional.

# 📌 Conclusão

A virtualização revolucionou a área da computação ao permitir múltiplos ambientes isolados dentro de um único hardware físico.

Além de reduzir custos e aumentar eficiência, essa tecnologia se tornou essencial para servidores, cloud computing, segurança e desenvolvimento de software.

Ferramentas como Oracle VirtualBox facilitaram o acesso à virtualização tanto em ambientes acadêmicos quanto profissionais.
